Payload CMS acquired by Figma in June 2025. Now has ~40.8k GitHub stars (fastest-growing headless CMS). Payload is embedded inside Next.js (one deployment), has auto-generated TypeScript types + GraphQL, jobs queue, field-level access control, Lexical rich text editor, live preview, and official templates (website, ecommerce).

**Why**: Figma acquisition accelerates Payload adoption via designer ecosystem. Payload + Figma = design-to-deployed pipeline. This is the most significant competitive development in headless CMS space.

**How to apply**: Sigil must go public (P0) before any P1 features matter. Counter-positioning: "Payload is for developers building one complex app. Sigil is for agencies managing many sites." Sigil wins on multi-tenancy (native RLS), Design Playground (no competitor has this), Resonance analytics (unique), vertical plugins (19), migration tools (WP/Shopify/Square), and pricing (more seats/sites per dollar).
