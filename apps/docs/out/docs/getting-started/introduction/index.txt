9:Tc4f,<h2>What is Sigil CMS?</h2>
<p>Sigil is a <strong>multi-tenant headless CMS framework</strong> built by Netrun Systems. It provides composable content blocks, a 19-plugin architecture, a Design Playground with 70+ Google Fonts, and an admin panel with visual editing.</p>
<blockquote>
<p><em>A sigil is a mark of identity and purpose -- exactly what your brand's web presence should be.</em></p>
</blockquote>
<p>Sigil is designed for:</p>
<ul>
<li><strong>Brand landing pages</strong> -- marketing sites with rich content blocks</li>
<li><strong>Artist sites</strong> -- music releases, events, streaming links, EPKs</li>
<li><strong>Service businesses</strong> -- booking, contact forms, mailing lists</li>
<li><strong>Documentation portals</strong> -- knowledge bases with search and feedback</li>
</ul>
<h2>Architecture Overview</h2>
<p>Sigil is built as a pnpm monorepo with Turborepo orchestration:</p>
<pre><code>packages/@netrun-cms/
  core/           -- Types, enums, Zod schemas, theme tokens, utilities
  db/             -- Drizzle ORM schema, migrations, client
  ui/             -- 64 Shadcn/Radix UI components + design system CSS
  theme/          -- ThemeProvider + 5 presets (netrun-dark, kog, intirkon, minimal, frost)
  blocks/         -- Composable content blocks with open registry
  embeds/         -- Platform embed components
  plugin-runtime/ -- Plugin loader, registry, manifest system

apps/
  api/            -- Express.js backend (port 3001 dev / 3000 prod)
  admin/          -- Vite + React 18 SPA (admin panel)

plugins/          -- 19 feature plugins (env-gated, graceful skip)
</code></pre>

<h2>Shared Libraries</h2>
<p>The API uses shared libraries from <code>@netrun/</code>:</p>
<ul>
<li><strong>@netrun/error-handling</strong> -- Error classes, correlation IDs, Express error handler</li>
<li><strong>@netrun/health</strong> -- <code>/health</code> and <code>/ready</code> endpoint factory</li>
<li><strong>@netrun/logger</strong> -- Pino structured logging + request middleware</li>
<li><strong>@netrun/security-middleware</strong> -- Helmet, CSRF, rate limiting</li>
</ul>
<h2>Design System</h2>
<ul>
<li><strong>Primary Color</strong>: <code>#90b9ab</code> (Netrun Sage Green)</li>
<li><strong>Typography</strong>: Futura Medium/Bold (headings), Inter (body)</li>
<li><strong>Dark mode</strong>: Class-based + <code>data-theme</code> attribute</li>
<li><strong>CSS Variables</strong>: Over 1,400 variables in the design system</li>
</ul>
