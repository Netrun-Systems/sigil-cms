9:Te6f,<h2>Prerequisites</h2>
<ul>
<li>Node.js >= 20</li>
<li>pnpm 9.15+</li>
<li>PostgreSQL database</li>
</ul>
<h2>Option 1: CLI Scaffolding</h2>
<p>The fastest way to get started:</p>
<pre><code class="language-bash"># Global install
npm install -g sigil-cms

# Create a new project
sigil create my-site
cd my-site
npm install
sigil dev
</code></pre>
<p>This creates a project with:</p>
<ul>
<li><code>package.json</code> with Sigil dependencies</li>
<li><code>sigil.config.ts</code> configuration file</li>
<li><code>.env.example</code> environment template</li>
<li><code>tsconfig.json</code> TypeScript configuration</li>
<li><code>content/</code>, <code>src/</code>, <code>public/</code> directories</li>
</ul>
<h2>Option 2: From the Monorepo</h2>
<p>For contributors or full-stack development:</p>
<pre><code class="language-bash">git clone https://github.com/Netrun-Systems/netrun-cms.git
cd netrun-cms
pnpm install
pnpm build
pnpm dev
</code></pre>
<p>This starts:</p>
<ul>
<li><strong>API</strong> on <code>http://localhost:3001</code></li>
<li><strong>Admin panel</strong> on <code>http://localhost:3001</code> (Vite dev)</li>
</ul>

<h2>Database Setup</h2>
<pre><code class="language-bash">cd packages/@netrun-cms/db
pnpm db:generate    # Generate migrations from schema
pnpm db:migrate     # Apply pending migrations
pnpm db:push        # Push schema directly (dev only)
pnpm db:studio      # Open Drizzle Studio UI
</code></pre>
<p>Or via the CLI:</p>
<pre><code class="language-bash">sigil migrate              # Apply pending migrations
sigil migrate --generate   # Generate from schema changes
sigil migrate --status     # Show migration status
</code></pre>
<h2>Seed Demo Content</h2>
<pre><code class="language-bash">sigil seed             # Populate with sample data
sigil seed --reset     # Drop existing data first
</code></pre>
<h2>Verify Installation</h2>
<pre><code class="language-bash">sigil info
</code></pre>
<p>This displays Node.js version, installed Sigil packages, config file status, and database connection status.</p>
