9:T2428,<h2>Base URL</h2>
<pre><code>https://your-instance.com/api/v1
</code></pre>
<h2>Authentication</h2>
<p>All authenticated endpoints require a JWT Bearer token:</p>
<pre><code>Authorization: Bearer &#x3C;your-jwt-token>
</code></pre>
<h2>Response Format</h2>
<p>All responses follow this shape:</p>
<pre><code class="language-json">{
  "success": true,
  "data": { ... },
  "meta": { "total": 100, "page": 1, "pageSize": 20 }
}
</code></pre>
<p>Error responses:</p>
<pre><code class="language-json">{
  "success": false,
  "error": {
    "code": "NOT_FOUND",
    "message": "Page not found"
  }
}
</code></pre>
