9:Ta3f,<h2>Overview</h2>
<p>The Store plugin adds full e-commerce capability to any Sigil site using Stripe for payment processing. It handles product catalogs, checkout sessions, order tracking, and Stripe webhooks.</p>
<p><strong>Required env</strong>: <code>STRIPE_SECRET_KEY</code>.</p>
<p>Uses the shared <code>@netrun/stripe-client</code> library from the Netrun TypeScript monorepo.</p>
<h2>Features</h2>
<ul>
<li><strong>Product catalog</strong> -- one-time and recurring products with Stripe sync</li>
<li><strong>Checkout sessions</strong> -- Stripe Checkout for secure payments</li>
<li><strong>Order tracking</strong> -- pending, paid, failed, refunded statuses</li>
<li><strong>Stripe webhooks</strong> -- automatic order status updates</li>
<li><strong>Line items</strong> -- per-order product breakdown</li>
</ul>
