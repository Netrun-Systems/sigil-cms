9:T8a5,<h2>Overview</h2>
<p>The Booking plugin adds appointment scheduling for service businesses. It supports configurable services, availability rules, and optional Google Calendar integration.</p>
<p><strong>Required env</strong>: None (Google Calendar is optional via <code>GOOGLE_CALENDAR_CREDENTIALS</code>).</p>
<h2>Features</h2>
<ul>
<li><strong>Service catalog</strong> -- define bookable services with duration, price, and buffer times</li>
<li><strong>Availability rules</strong> -- per-day-of-week time slots with timezone support</li>
<li><strong>Booking management</strong> -- track appointments by status (pending, confirmed, cancelled)</li>
<li><strong>Google Calendar sync</strong> -- automatically create calendar events (optional)</li>
<li><strong>Email notifications</strong> -- confirmation and reminder emails via ACS (optional)</li>
</ul>
