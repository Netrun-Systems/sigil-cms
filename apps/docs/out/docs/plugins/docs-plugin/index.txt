9:Tc08,<h2>Overview</h2>
<p>The Docs plugin turns Sigil into a documentation portal. Articles link to CMS pages (using the block editor for content), and the plugin adds KB-specific metadata, categorization, revision tracking, and a "Was this helpful?" feedback system.</p>
<p><strong>Required env</strong>: None.</p>
<h2>Features</h2>
<ul>
<li><strong>Article categories</strong> -- hierarchical categories with slug, icon, and sort order</li>
<li><strong>Full-text search</strong> -- search articles by title, excerpt, and tags</li>
<li><strong>Version history</strong> -- track revisions with change notes and version numbers</li>
<li><strong>Reader feedback</strong> -- "Was this helpful?" with optional comments</li>
<li><strong>View tracking</strong> -- count article views</li>
<li><strong>Featured and pinned articles</strong> -- highlight important content</li>
</ul>
