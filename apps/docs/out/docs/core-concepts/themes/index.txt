9:Te1d,<h2>Theme Model</h2>
<p>Themes define the visual identity of a site through design tokens. Each site can have multiple themes with one active at a time.</p>
<h3>Theme Properties</h3>
<table>
<thead>
<tr>
<th>Field</th>
<th>Type</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>name</code></td>
<td>string</td>
<td>Theme name</td>
</tr>
<tr>
<td><code>baseTheme</code></td>
<td>string</td>
<td>Preset: <code>netrun-dark</code>, <code>kog</code>, <code>intirkon</code>, <code>minimal</code>, <code>frost</code></td>
</tr>
<tr>
<td><code>isActive</code></td>
<td>boolean</td>
<td>Whether this is the active theme</td>
</tr>
<tr>
<td><code>tokens</code></td>
<td>object</td>
<td>Design tokens (colors, typography, spacing, effects)</td>
</tr>
<tr>
<td><code>customCss</code></td>
<td>string</td>
<td>Additional custom CSS</td>
</tr>
</tbody>
</table>

<h2>Theme API</h2>
<p>All routes are under <code>/api/v1/sites/:siteId/themes</code>.</p>
<h3>List themes</h3>
<pre><code>GET /api/v1/sites/:siteId/themes
</code></pre>
<p>Query params: <code>page</code>, <code>limit</code>, <code>baseTheme</code>, <code>isActive</code>.</p>
<h3>Get active theme</h3>
<pre><code>GET /api/v1/sites/:siteId/themes/active
</code></pre>
<h3>Create a theme</h3>
<pre><code>POST /api/v1/sites/:siteId/themes
</code></pre>
<h3>Activate a theme</h3>
<pre><code>POST /api/v1/sites/:siteId/themes/:id/activate
</code></pre>
<p>Deactivates all other themes for this site.</p>
<h3>Duplicate a theme</h3>
<pre><code>POST /api/v1/sites/:siteId/themes/:id/duplicate
</code></pre>
<pre><code class="language-json">{ "name": "My Theme Copy" }
</code></pre>
<h3>Public theme</h3>
<p>Consumer sites fetch the active theme without authentication:</p>
<pre><code>GET /api/v1/public/sites/:siteSlug/theme
</code></pre>
<h2>Design Playground</h2>
<p>The admin panel includes a visual Design Playground with:</p>
<ul>
<li><strong>70+ Google Fonts</strong> -- searchable font browser</li>
<li><strong>Custom font upload</strong> -- <code>.woff2</code>, <code>.ttf</code>, <code>.otf</code> files</li>
<li><strong>Button shapes</strong> -- rounded, pill, square, with size variants</li>
<li><strong>Spacing controls</strong> -- section padding, block gaps</li>
<li><strong>Shadow presets</strong> -- subtle, medium, dramatic</li>
<li><strong>Glass effects</strong> -- backdrop blur with opacity</li>
</ul>
