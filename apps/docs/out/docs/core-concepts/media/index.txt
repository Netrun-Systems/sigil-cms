9:Tbac,<h2>Overview</h2>
<p>Sigil's media library manages files with folder organization, search, and metadata. It supports single and bulk uploads (up to 20 files, 50MB each).</p>
<h2>Media Endpoints</h2>
<p>All media routes are scoped to a site: <code>/api/v1/sites/:siteId/media</code>.</p>
<h3>Upload a file</h3>
<pre><code>POST /api/v1/sites/:siteId/media/upload
Content-Type: multipart/form-data
</code></pre>
<p>Form fields:</p>
<ul>
<li><code>file</code> -- the file to upload (required)</li>
<li><code>altText</code> -- accessibility text (optional)</li>
<li><code>caption</code> -- display caption (optional)</li>
<li><code>folder</code> -- folder path (optional, e.g., <code>images/hero</code>)</li>
</ul>
<h3>Bulk upload</h3>
<pre><code>POST /api/v1/sites/:siteId/media/upload/bulk
Content-Type: multipart/form-data
</code></pre>
<p>Form fields:</p>
<ul>
<li><code>files</code> -- up to 20 files</li>
<li><code>folder</code> -- shared folder path</li>
</ul>
<h3>List media</h3>
<pre><code>GET /api/v1/sites/:siteId/media
</code></pre>
<p>Query params:</p>
<ul>
<li><code>page</code>, <code>limit</code> -- pagination</li>
<li><code>folder</code> -- filter by folder path</li>
<li><code>mimeType</code> -- filter by type (e.g., <code>image</code>, <code>image/png</code>, <code>video</code>)</li>
<li><code>search</code> -- search filename, original filename, alt text</li>
</ul>
<h3>List folders</h3>
<pre><code>GET /api/v1/sites/:siteId/media/folders
</code></pre>
<p>Returns all folders with file counts.</p>
<h3>Create media record (manual)</h3>
<pre><code>POST /api/v1/sites/:siteId/media
</code></pre>
<pre><code class="language-json">{
  "filename": "hero-bg.jpg",
  "originalFilename": "Hero Background.jpg",
  "mimeType": "image/jpeg",
  "fileSize": 245760,
  "url": "https://storage.example.com/hero-bg.jpg",
  "thumbnailUrl": "https://storage.example.com/hero-bg-thumb.jpg",
  "altText": "Mountain landscape at sunset",
  "folder": "images/hero",
  "metadata": {}
}
</code></pre>
<h3>Update metadata</h3>
<pre><code>PUT /api/v1/sites/:siteId/media/:id
</code></pre>
<p>Update <code>altText</code>, <code>caption</code>, <code>folder</code>, or <code>metadata</code>.</p>
<h2>Storage Providers</h2>
<p>The Photos plugin (<code>AZURE_STORAGE_CONNECTION_STRING</code>) enables Azure Blob Storage for file uploads. When configured, the Gemini AI can automatically generate alt text and curate photos.</p>
<p>Without the Photos plugin, media records are stored as URL references to externally-hosted files.</p>
